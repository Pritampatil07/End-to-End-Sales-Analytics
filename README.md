# End-to-End Sales Analytics

## Project Overview
This project demonstrates end-to-end sales analytics using Excel, Power BI, and SQL.

The dashboard provides insights into:
- Sales Performance
- Profit Analysis
- Customer Segmentation
- Regional Performance
- Product Analysis
- Sales Forecasting

## Tools & Technologies
- Power BI
- Microsoft Excel
- DAX
- Power Query
- Data Modeling
- SQL

## Dataset
Sample Superstore Dataset

## Key KPIs
- Total Sales: 2.30M
- Total Profit: 286K
- Profit Margin: 12.47%
- Forecast Accuracy: 95%

## Dashboard Pages
1. Executive Overview
2. Product Analysis
3. Customer Analysis
4. Regional Analysis
5. Sales Trend & Forecast
6. Excel Dashboard

## Business Insights
- Technology category generates the highest sales.
- Copiers contribute the highest profit.
- Consumer segment contributes the largest share of sales.
- Sales and profit show consistent growth over time.
- Forecast accuracy indicates reliable future projections.
